# Scene Transitions Plugin

Scene transition system with shader-based wipe/fade patterns.

## Features

- Centralized transition controller
- Multiple shader-based transition effects
- Simple API for switching scenes with transitions

## Installation

1. Copy `scene_transitions` to your project's `addons/` folder
2. Enable **Scene Transitions** in **Project → Project Settings → Plugins**

## Usage

- Use the provided transition scripts/nodes to change scenes with a chosen effect
- Designed to work alongside RPG Entities and other gsg plugins
