# Multimedia Zones Plugin

Area3D zones that control music and camera behavior with priorities.

## Features

- Zone-based control for Music Manager and Camera System
- Priority-based blending when multiple zones overlap
- Auto-configuration of cameras for zones (when needed)

## Installation

1. Copy `multimedia_zones` to your project's `addons/` folder (or use this submodule)
2. Enable **Multimedia Zones** in **Project → Project Settings → Plugins**

## Usage

- Place `MultimediaZone` nodes in your scenes to drive music/camera changes
- Works together with **Music Manager** and **Camera System** for dynamic audio/visual behavior
